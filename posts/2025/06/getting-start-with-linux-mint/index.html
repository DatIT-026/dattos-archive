Getting Start with Linux Mint

Hello there! I’m Datto and today, this post will guide you on how to use Linux Mint. If you don’t know what Linux is or don’t know how to install it. Learn about it in Part 1.

Ready or not? Let us start.

  1. Cinnamon
  2. Sway
  3. Conclusion

What is Cinnamon!

Now, before we dive in. Let me introduce you what a Desktop environment is. A desktop environment on Linux is the graphical interface you see and use, like windows, icons, and menus. It makes using Linux easier by letting you interact with your computer visually instead of typing commands. Each desktop environment offers various styles and features, so you can pick the one that suit your preferences and hardware.

“The most popular version of Linux Mint is the Cinnamon edition. Cinnamon is primarily developed for and by Linux Mint. It is slick, beautiful, and full of new features.” is what the main page of Linux Mint says. Cinnamon is super friendly interface for users who are new to the Linux world.

The first thing you need to do after booting into Cinnamon for the first time is to update your system through the Update Manager.

  • Open the Update Manager from the menu.
  • Click the Refresh button to check for available updates.
  • Select the updates you want to install.
  • Click Install Updates and enter your password if prompted.
  • Wait for the updates to finish, then restart your computer if needed.

This keeps your system secure and up to date easily.

After updating your system in Update Manager, the next steps is check for and install any additional drivers (like graphics or Wi-Fi) via the Driver Manager.

Great! Everything is set up. Now, you can customize your theme or install the software and packages you need.

Sway is also good to use

If you want a different experience from Windows or traditional desktop environments, try using Sway which is a modern tiling window manager and Wayland compositor.

But wait! What is Wayland?

First you need to know what is Display Server. This is the low-level system that talks to your computer’s graphics hardware and shows things on the screen. It’s like the engine that powers the display.

Wayland and its older system called X11 are systems that help your computer display graphics on the screen.

  • X11 is the older system, widely used for decades. It handles drawing windows and graphics but can be complex and less efficient.

  • Wayland is a newer, simpler system designed to be faster, more secure, and smoother, improving how graphics and windows are managed.

And… What is tiling window manager?

A tiling window manager is a type of window manager that automatically arranges your open windows side by side without overlapping. It helps you work faster by keeping everything organized and easy to navigate, mostly using the keyboard. There are various tiling window managers on Linux such as Hyprland, sway, i3, and more.

How to install sway

Open the terminal in the menu and enter this code:

sudo apt update
sudo apt install sway

If you are using NVDIA gpu or any unsupported gpu, modify the session file to enable compatibility. Edit the file in /usr/share/wayland-sessions/ and change the line staring with Exec= to

Exec=sway --unsupported-gpu

This tells Sway to run with options that help it work better with NVIDIA or other unsupported GPUs, since NVIDIA’s Wayland support is still limited and can cause issues without this workaround.

Next, save your work and log out from your current Cinnamon session.

At the login screen, look for a session or gear icon. Click it and select Sway as your session. Then enter your password and login it. You can always log out and select Cinnamon again if you want to switch back.

Conclusion

Congratulation! Now you understand some important concepts:

  • Display Server (e.g. X11 or Wayland): This is the low-level system that talks to your computer’s graphics hardware and shows things on the screen. It’s like the engine that powers the display.
  • Window Manager: This controls how windows look and behave, including how they open, close, move, and resize. It arranges your windows on the screen.
  • Desktop Environment (like Cinnamon or XFCE): This is the complete user interface you see and interact with. It includes the window manager plus menus, panels, icons, and apps to make your desktop easy and friendly to use.

You have not only set up your desktop environment (Cinnamon) for the first time but also learned how to install and log into Sway, a tiling window manager.
