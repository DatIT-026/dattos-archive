Setup and Fix Windows Tool (SAFW)

[⚠ NOTE] This project will be retired on December 31st 2026.

A streamlined command-line utility for quick access to Windows settings and troubleshooting common issues. Key features include:

  • System Configuration: Access Computer Management, System Properties, and Programs.

  • Network & Devices: Manage Network, Sharing, and Print options.

  • Customization: Modify Personalization and Power settings.

  • Troubleshooting Tools: Fix Windows Updates, Wi-Fi issues, and view activation status.

  • Utilities: Date/Time settings, Region, System Info, Activate Windows and more.

Designed for IT support, this tool is user-friendly and organized for fast system management.

Ensure to run as Administrator for optimal functionality.
